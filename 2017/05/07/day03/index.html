c自学Day03

摘要：c/c++ 自学第三天

数据结构

研究节点和节点之间的关系 (数组/链表/树/图)

数据逻辑结构：
集合： 散列
线性结构：一对一
树形结构：一对多
图形结构：多对多

数据的物理结构或存储结构：
顺序
链式
索引
散列

算法：特定问题求解步骤的描述

算法与数据结构的关系：
程序 = 数据结构 + 算法

算法的实现要编译成具体的计算机指令
每一个指令 在具体的计算机 cpu 上运行的时间固定
通过具体的步骤就可以推导出算法的复杂度

算法复杂度表示：
2n + 1 -》 o(n)
2 -》o(1)
