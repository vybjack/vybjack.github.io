搭建博客

摘要：基于 hexo 和 github 搭建个人免费博客

关于博客

今天花了几个小时搭建了自己的个人博客，就想着把怎样搭建博客的过程记录下来，以便以后查阅。本次搭建以mac系统为例。

准备工具

· Git
· Node.js
需要安装上述两个工具，打开官网下载、安装即可

终端安装 hexo

终端创建本地博客文件内容

预览本地博客地址

如果搭建顺利，浏览器打开http://localhost:4000 即为本地的博客首页，到这里我们的博客算是搭建好了，不过只是本地的。所以，接下来我们要提交到网络，这里我们利用github

关联 github

注意：每个属性冒号后面都必须带有空格，其中repository的值中nickname改成需要关联的 github 的帐号昵称

终端安装 hexo-deployer-git

终端将本地博客内容提交到 github

hexo 一次部署： hexo clean –> hexo generate –> hexo deploy

博客搭建完成

如果顺利，浏览器打开 https://nickname.github.io (nickname为关联的 github 的昵称) 即为搭建好的博客，到此博客搭建完成
